# Reveal True Name (8)

**Source:** CB77015 - Demons of Doom, Page 61

**Page Image:** [View Page 61](../../../../CB77015%20-%20Demons%20of%20Doom_Pages/Demonology/page_0061.png)

**Rank:** 8

**Magic Point Cost:** 8

## Description

This powerful spell will allow a Demonologist to find out the true name of a Demon.

## Requirements

- The Demon must be within 10 feet for the spell to work
- The Demon must also fail a Test for LUCK (or SKILL)

## Effects

Once known, the Demonologist gains various advantages when attempting to control this specific Demon and can also summon it using a Ritual.
