# Detect Demon (1)

**Source:** CB77015 - Demons of Doom, Page 60

**Page Image:** [View Page 60](../../../../CB77015%20-%20Demons%20of%20Doom_Pages/Demonology/page_0060.png)

**Rank:** 1

**Magic Point Cost:** 1

## Description

Casting this spell reveals the exact location of every Demon within 50 feet, even if not visible. This knowledge applies only to that specific instance of casting.
