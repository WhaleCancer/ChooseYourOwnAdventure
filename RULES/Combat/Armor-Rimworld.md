# Armor - Rimworld

**Note:** Statistics have been converted from Rimworld's armor system to Advanced Fighting Fantasy protection track system. Original Rimworld values (sharp/blunt/heat protection percentages) are provided in individual armor files for reference. Rimworld uses percentage-based protection against different damage types (sharp, blunt, heat). These have been converted to AFF's 1-7+ protection track format.

## Core Game Armor

For detailed armor statistics, refer to the individual armor files or the [RimWorld Wiki](https://rimworld.fandom.com/wiki/Armor).
