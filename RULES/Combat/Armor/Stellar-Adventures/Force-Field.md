## Notes

A personal hi-tech shield uses up 1 charge each time it actually stops damage. If the shield fails to stop any damage, a charge is not used. The shield will not function at all with an empty power pack and it takes a full round to reload the unit. Some require a free hand (cannot use with rifles/heavy weapons). Protection values are added to armor worn. When using both armor and a shield, a single "Armour" roll is made, both tables are consulted, and the highest value is used.
