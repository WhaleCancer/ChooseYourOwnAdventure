- **Special:** Shoulder-mounted grenade launcher

## Description

Marine armor equipped with a shoulder-mounted grenade launcher. Provides the same excellent protection as standard marine armor while adding the ability to launch grenades without using hands. Allows the wearer to fire grenades while using other weapons or performing actions. Requires Royalty DLC and spacer technology.
