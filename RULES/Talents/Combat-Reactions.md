# Combat Reactions

**Source:** Core rulebook (AF2e)

**Cost:** 4 Talent Points

This Talent grants supernatural awareness and control in battle, providing two abilities:

1. **Cannot be Ambushed:** The Character cannot be ambushed (the ambusher gains no combat benefits).

2. **Outnumbered Defense:** The Character suffers fewer penalties when outnumbered. If outnumbered 1-3 to one, attackers gain no benefit. If outnumbered 4 to one, attackers gain a +1 bonus; 5 to one, +2; and 6 to one, +3.
