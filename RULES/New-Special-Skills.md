# New Special Skills

These new Special Skills can be taken by Heroes at creation (with the Director's approval) or learned later. Existing Heroes can learn these new Special Skills using normal rules.

**Source:** Heroes Companion (Chapter 1)

## New Combat Special Skills

Combat Skills add to PHYSICAL for that specific test.

- [Battle Tactics](Special-Skills/Combat-Skills/Battle-Tactics.md)
- [Siege Weapons](Special-Skills/Combat-Skills/Siege-Weapons.md)
