# World Lore

**Characteristic:** MENTAL

**Tech Levels:** Basic, Low, Medium, High

This skill grants a Character broad knowledge about the world they live in. It provides common information about nations, natural features (seas, mountain ranges, forests), common cultural features, and some history. It doesn't replace other skills like "Forest Lore" but allows the Character to at least know the location and reputation of a specific large forest.
